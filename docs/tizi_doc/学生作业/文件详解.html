student_paper_base.php

protected function _getQuestionsByPid($paper_id)

根据paper_id 获取题目 并对题目内容处理 ， 比如 过滤标签，
区分线上作业题目 跟 线下作业题目(线上作业题目的判断依据为: question_type 为 3， 且asw必须为 ['A','B','C','D'] 中其中一项)

protected function _get_question_order($random_num, $question_num) 获取题目顺序，教师布置作业如果选中打乱顺序, 会通过此方法获取大乱后的题目顺序.
protected function _question_restructing($questions, $question_order) 根据提供的题目内容跟新题目顺序，重新组合题目, 返回打乱后的题目内容
